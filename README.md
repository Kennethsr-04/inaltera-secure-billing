## 💎 ¿Por qué Inaltera?

**Inaltera** nace bajo la premisa de la inmutabilidad. No es solo una app de facturación; es un registro blindado para tu negocio. 

> *"Lo que se escribe en Inaltera, permanece en Inaltera."*

---

## 🚀 Características (El Inventario)

* **🔒 Cifrado de Grado Bancario:** Cada factura se sella con una firma digital única.
* **📊 Dashboards de Cristal:** Visualiza tus ingresos con claridad absoluta.
* **📑 Plantillas Inteligentes:** Crea facturas profesionales.
* **☁️ Sincronización en la Nube:** Tus datos, accesibles pero intocables por terceros.

## ⚙️ Protocolo de Instalación

Sigue estos pasos para desplegar tu propia bóveda de facturación:

1. **Clonar el repositorio:**
   ```bash
   git clone [https://github.com/tu-usuario/inaltera.git](https://github.com/tu-usuario/inaltera.git)

2. ✅ **Descargar el ZIP del proyecto:**
   Descarga y extrae el ZIP del proyecto y abre la carpeta en tu entorno de desarrollo.

   💡 **Usando Visual Studio Code:** 
   * En la consola de comandos escribir:
   * 1 - "npm install"
   * 2 - "npm run dev"
   * 3 - Abrir en local
